Задача

Разработать визуальную концепцию и дизайн лендинга для музыкального проекта «МИСАКО». Клиенту нужно было показать два лица: универсальную кавер-группу для мероприятий и аутентичный арт-проект с собственной музыкой. Главная сложность — объединить восточную эстетику с современным веб-дизайном, не скатываясь в откровенные стереотипы.

Решение

Дизайн построен на контрасте тёмной, атмосферной эстетики и ярких акцентов (красный, золотой). Основные инструменты:

  • глубокая типографика с иероглифическими акцентами
  • лаконичные карточки для разделов «Анонсы», «Билеты», «Мерч»
  • два смысловых блока для двух форматов группы (кавер / арт-проект)
  • адаптивная сетка, подчёркивающая восточный минимализм

Цель — не просто показать информацию, а передать атмосферу: восточная медитативность, инструментальную текстуру и сюжеты из азиатской мифологии. Сайт должен говорить с аудиторией на языке образов, а не фактов.

Страница арт-проекта

Для собственной музыки «МИСАКО» потребовалась более глубокая, мифологическая эстетика. Я использовала:

  • приглушённую цветовую палитру с акцентами тёмного золота
  • крупные образы (традиционные инструменты, силуэты)
  • типографику с «нарративным» ритмом — тексты как часть композиции

Задача была — не просто показать группу, а создать ощущение, что пользователь заходит в другую реальность, где древние мифы звучат через современный дизайн.

Моя роль

дизайнер концепция, визуальная система, макеты страниц, адаптивная вёрстка, подготовка ассетов.
