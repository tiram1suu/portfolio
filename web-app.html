Веб-приложение

для Центра российской истории и культуры — ВШЭ
2025 дизайн / frontend / Telegram Web App

Задача

Разработать веб-приложение для Центра российской истории и культуры при факультете креативных индустрий ВШЭ. Основная цель — создать платформу, где можно легко публиковать события, а пользователи — быстро записываться и отменять записи.

Также требовалась геймификация: за посещение событий пользователь получает «монетки», которые в будущем можно будет потратить на что-то внутри системы.

Процесс работы

Я полностью спроектировала интерфейс и реализовала его на HTML, CSS и JavaScript. Работа велась по чёткому графику, с акцентом на пользовательские сценарии и тестирование.

  • Дни 1–5: анализ, user flow, wireframes
  • Дни 6–12: UI-дизайн, прототип, дизайн-система
  • Дни 13–18: реализация интерфейса, адаптивность, интерактивность, финальное тестирование

Результат — полностью готовое к запуску веб-приложение, интегрированное в Telegram как Web App.

Мокапы

Главный экран, список событий, профиль пользователя с монетками и достижениями.

Ключевые решения

  • Простая система записи: в один клик — запись на событие, повторный клик — отмена.
  • Геймификация через монеты: каждое посещение приносит монетки, которые накапливаются в профиле.
  • Достижения: система поощрения за активность (первое событие, посещение событий разных типов).
  • Telegram Web App: приложение полностью адаптировано под встраивание в Telegram с нативной навигацией.

Моя роль

дизайнер frontend-разработчик концепция, user flow, UI/UX, прототип, вёрстка (HTML/CSS/JS), интеграция с Telegram Web App.
